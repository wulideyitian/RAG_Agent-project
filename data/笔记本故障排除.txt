一、开机与电源故障（20 条）
故障现象：开机无反应，指示灯不亮；检测：电源适配器是否插紧、电源线是否破损，电池是否安装到位；修复：更换完好电源线，重新插拔电池，确保充电座通电，仍无反应联系售后检测主板。
故障现象：开机后立即自动关机；检测：电池电量是否耗尽，机身是否过热，电源适配器是否故障；修复：充满电后重试，移至通风处冷却，更换适配电源适配器。
故障现象：开机黑屏，风扇转但无显示；检测：屏幕排线是否松动，显卡是否故障，外接显示器能否显示；修复：重新插拔屏幕排线（需专业操作），外接显示器确认后联系售后修显卡。
故障现象：开机蓝屏，显示错误代码；检测：系统文件是否损坏，内存是否故障，硬盘是否有坏道；修复：用 PE 系统修复系统文件，拔插内存，用 DiskGenius 检测硬盘坏道。
故障现象：开机进 BIOS，无法进入系统；检测：硬盘是否识别，启动项是否设置正确；修复：在 BIOS 中确认硬盘识别，设置硬盘为第一启动项，重建 MBR 引导。
故障现象：电源适配器插电后指示灯不亮；检测：适配器是否损坏，插座是否通电；修复：更换插座，用万用表检测适配器输出电压，损坏则更换。
故障现象：电池充不进电，指示灯不亮；检测：电源适配器是否通电，充电触点是否氧化，电池是否损坏；修复：更换电源适配器，用橡皮擦擦拭氧化触点，电池损坏则更换。
故障现象：开机后反复重启，无法进入系统；检测：系统是否中毒，硬件驱动是否冲突，CPU 温度是否过高；修复：进入安全模式杀毒，卸载冲突驱动，清理散热口灰尘。
故障现象：笔记本自动休眠，无法唤醒；检测：休眠设置是否有误，电源管理驱动是否正常；修复：调整电源休眠设置，更新电源管理驱动，关闭快速启动。
故障现象：开机后键盘无反应，外接键盘正常；检测：键盘排线是否松动，键盘驱动是否损坏；修复：重新插拔键盘排线，卸载并重装键盘驱动。
故障现象：电源指示灯闪烁，无法正常充电；检测：充电触点是否有灰尘、氧化，电池是否鼓包；修复：用干布擦拭充电触点，更换鼓包电池。
故障现象：开机后屏幕花屏、有条纹；检测：显卡驱动是否过时，屏幕是否损坏，排线是否松动；修复：更新显卡驱动，外接显示器排查，损坏则联系售后换屏幕。
故障现象：笔记本无法开机，电池有电；检测：电源按键是否故障，主板是否短路；修复：检查电源按键，联系售后检测主板。
故障现象：开机后提示 “找不到操作系统”；检测：硬盘是否松动，系统是否崩溃；修复：重新插拔硬盘，用 U 盘启动盘重装系统。
故障现象：电源适配器发热严重，有异味；检测：适配器是否过载，是否为原装；修复：立即停止使用，更换原装适配器，避免短路。
故障现象：开机后风扇狂转，噪音大；检测：散热口是否堵塞，CPU 温度是否过高；修复：清理散热口灰尘，更换散热硅脂。
故障现象：笔记本开机后无声音，外接耳机正常；检测：内置扬声器是否损坏，音频驱动是否正常；修复：更新音频驱动，联系售后检测扬声器。
故障现象：开机后触控板无反应；检测：触控板驱动是否损坏，是否被快捷键禁用；修复：更新触控板驱动，按 Fn + 触控板快捷键启用。
故障现象：笔记本自动断电，无任何提示；检测：CPU 温度是否过高，电池是否老化；修复：清理散热系统，更换老化电池。
故障现象：开机后提示 “硬盘错误”；检测：硬盘是否有坏道，数据线是否松动；修复：用 DiskGenius 修复坏道，重新插拔硬盘数据线。
二、性能与散热故障（20 条）
故障现象：笔记本运行卡顿，反应缓慢；检测：内存是否不足，硬盘是否碎片化，后台程序是否过多；修复：增加内存，磁盘碎片整理，关闭多余后台程序。
故障现象：游戏 / 设计软件运行卡顿，帧率低；检测：显卡驱动是否过时，散热是否不足，电源模式是否为节能；修复：更新显卡驱动，清理散热灰尘，切换至高性能模式。
故障现象：CPU 占用率长期 100%；检测：是否有病毒进程，驱动是否冲突，后台服务是否过多；修复：杀毒，卸载冲突驱动，禁用不必要的后台服务。
故障现象：笔记本机身发热严重，有异味；检测：散热口是否堵塞，风扇是否故障，导热硅脂是否干涸；修复：清理散热口，更换故障风扇，重新涂抹导热硅脂。
故障现象：硬盘读写速度骤降；检测：硬盘是否有坏道，缓存是否耗尽，是否开启节能模式；修复：修复硬盘坏道，关闭节能模式，更换老化硬盘。
故障现象：笔记本频繁蓝屏，代码指向内存；检测：内存是否松动，内存是否不兼容，内存是否损坏；修复：拔插内存，更换兼容内存，用 MemTest 测试内存。
故障现象：外接显示器卡顿，本机屏幕正常；检测：视频接口是否松动，显卡外接输出是否故障；修复：重新插拔视频线，更新显卡驱动，联系售后检测显卡。
故障现象：笔记本运行时突然降频，性能下降；检测：CPU 温度是否过高，电源适配器功率是否不足；修复：清理散热，更换原装高功率适配器。
故障现象：固态硬盘掉速严重；检测：是否开启 SLC 缓存，硬盘是否过热，是否为 QLC 闪存；修复：关闭节能模式，散热降温，更换 TLC 闪存硬盘。
故障现象：笔记本休眠后唤醒卡顿；检测：内存是否不足，休眠文件是否损坏；修复：增加内存，删除休眠文件重新生成。
故障现象：CPU 温度正常但性能不足；检测：电源模式是否设置正确，BIOS 是否限制性能；修复：切换至高性能模式，更新 BIOS 解除性能限制。
故障现象：笔记本运行大型软件时自动退出；检测：内存是否不足，显卡是否驱动异常，软件是否兼容；修复：增加内存，更新显卡驱动，安装兼容版本软件。
故障现象：硬盘占用率 100%，系统卡顿；检测：是否有后台程序读写硬盘，硬盘是否老化；修复：关闭后台读写程序，更换高速硬盘。
故障现象：笔记本风扇不转，机身过热；检测：风扇是否被灰尘堵塞，风扇驱动是否故障；修复：清理风扇灰尘，更换故障风扇。
故障现象：显卡驱动频繁崩溃；检测：驱动是否过时，显卡是否过热，电源是否稳定；修复：回滚至稳定驱动版本，清理显卡散热，检查电源稳定性。
故障现象：笔记本运行时出现 “滋滋” 异响；检测：硬盘是否故障，风扇是否卡顿，电源适配器是否异常；修复：检测硬盘坏道，清理风扇，更换适配器。
故障现象：内存不足提示，实际内存充足；检测：内存是否被占用，虚拟内存是否设置过小；修复：关闭占用内存程序，增大虚拟内存。
故障现象：笔记本外接设备后性能下降；检测：外接设备是否耗电过多，USB 接口是否故障；修复：拔掉不必要的外接设备，更换 USB 接口。
故障现象：笔记本电池续航大幅衰减，性能下降；检测：电池是否老化，是否开启节能模式；修复：更换电池，关闭过度节能设置。
故障现象：笔记本运行时突然黑屏，电源灯亮；检测：显卡是否故障，屏幕是否损坏；修复：强制重启，外接显示器排查，联系售后检测显卡和屏幕。
三、网络与连接故障（20 条）
故障现象：笔记本无法连接 WiFi；检测：WiFi 开关是否开启，密码是否正确，路由器是否正常；修复：开启 WiFi 开关（Fn+F5），重新输入密码，重启路由器。
故障现象：WiFi 连接后频繁断开；检测：WiFi 信号是否薄弱，是否有信号干扰，驱动是否过时；修复：移至信号强区域，避开干扰源，更新无线网卡驱动。
故障现象：笔记本无法识别 WiFi 网络；检测：WiFi 网卡是否故障，路由器是否支持该频段；修复：更新网卡驱动，确认路由器支持 2.4G/5G，重启网卡。
故障现象：有线网络无法连接；检测：网线是否破损，网口是否松动，路由器 LAN 口是否正常；修复：更换网线，重新插拔网线，更换路由器 LAN 口。
故障现象：网络速度慢，下载卡顿；检测：网络带宽是否被占用，WiFi 信号是否干扰，网卡驱动是否过时；修复：关闭占用带宽程序，切换至 5G WiFi，更新网卡驱动。
故障现象：笔记本无法连接蓝牙设备；检测：蓝牙是否开启，设备是否配对，驱动是否正常；修复：开启蓝牙（Fn+F6），重新配对设备，更新蓝牙驱动。
故障现象：蓝牙设备连接后断连；检测：蓝牙信号是否被遮挡，设备距离是否过远，驱动是否冲突；修复：移除遮挡物，缩短设备距离，卸载冲突驱动。
故障现象：笔记本无法识别外接 USB 设备；检测：USB 接口是否故障，设备是否损坏，驱动是否缺失；修复：更换 USB 接口，测试设备在其他电脑，安装对应驱动。
故障现象：USB 设备连接后无法使用；检测：USB 接口供电是否不足，设备是否兼容，驱动是否正常；修复：连接后置 USB 接口（供电足），更换兼容设备，更新驱动。
故障现象：笔记本外接显示器无信号；检测：视频线是否破损，接口是否松动，显示器是否开启；修复：更换视频线，重新插拔接口，开启显示器并切换输入源。
故障现象：HDMI 接口无输出；检测：HDMI 线是否损坏，接口是否松动，显卡驱动是否过时；修复：更换 HDMI 线，重新插拔，更新显卡驱动。
故障现象：笔记本无法连接投影仪；检测：投影模式是否正确，接口是否匹配，驱动是否正常；修复：切换投影模式（Win+P），使用适配接口线，更新显卡驱动。
故障现象：无线网卡无法启用；检测：网卡是否被禁用，驱动是否损坏，硬件是否故障；修复：在设备管理器启用网卡，更新驱动，联系售后检测硬件。
故障现象：笔记本连接 WiFi 后无法上网；检测：路由器是否联网，DNS 是否设置错误；修复：检查路由器联网状态，设置 DNS 为 223.5.5.5/119.29.29.29。
故障现象：蓝牙无法搜索到设备；检测：蓝牙可见性是否开启，设备是否进入配对模式；修复：开启蓝牙可见性，让设备进入配对模式，重启蓝牙。
故障现象：外接耳机无声音；检测：耳机是否损坏，音频接口是否故障，音量是否静音；修复：测试耳机在其他设备，更换音频接口，关闭静音（Fn+F11）。
故障现象：笔记本内置麦克风无声音；检测：麦克风是否被禁用，驱动是否正常，音量是否过低；修复：在声音设置启用麦克风，更新音频驱动，调大音量。
故障现象：笔记本连接打印机无法打印；检测：打印机是否联网，驱动是否安装，连接是否正常；修复：确认打印机联网，安装对应驱动，重新连接 USB / 网络。
故障现象：Type-C 接口无法充电 / 传输数据；检测：接口是否松动，线材是否支持，驱动是否正常；修复：重新插拔 Type-C 线，使用全功能 Type-C 线，更新接口驱动。
故障现象：笔记本连接热点后无法上网；检测：热点是否联网，密码是否正确，网络设置是否错误；修复：确认热点联网，重新输入密码，重置网络设置（cmd 输入 netsh winsock reset）。
四、软件与系统故障（20 条）
故障现象：系统无法正常启动，提示文件丢失；检测：系统文件是否损坏，是否误删系统文件；修复：用 U 盘启动盘修复系统，重建系统文件，无法修复则重装系统。
故障现象：软件安装失败；检测：软件是否兼容系统，是否有旧版本残留，权限是否足够；修复：安装兼容版本，卸载旧版本残留，以管理员身份运行安装包。
故障现象：软件运行时崩溃；检测：软件是否兼容，驱动是否过时，内存是否不足；修复：安装兼容版本，更新显卡 / 声卡驱动，增加内存。
故障现象：系统蓝屏，提示 “INACCESSIBLE_BOOT_DEVICE”；检测：硬盘是否松动，硬盘是否有坏道，BIOS 设置是否错误；修复：重新插拔硬盘，修复硬盘坏道，恢复 BIOS 默认设置。
故障现象：系统卡顿，开机时间过长；检测：开机启动项过多，硬盘是否碎片化，系统是否中毒；修复：禁用不必要启动项，磁盘碎片整理，全盘杀毒。
故障现象：桌面图标消失；检测：是否隐藏图标，系统是否故障，是否误操作删除；修复：显示桌面图标（右键桌面 - 查看 - 显示桌面图标），重启电脑，重建图标缓存。
故障现象：任务栏无法点击，无响应；检测：系统是否卡顿，任务栏是否冻结，驱动是否冲突；修复：重启资源管理器（任务管理器 - 结束 explorer.exe - 新建任务），更新驱动。
故障现象：系统提示 “内存不足”，实际内存充足；检测：虚拟内存是否设置过小，是否有内存泄漏程序；修复：增大虚拟内存，关闭内存泄漏程序，重启电脑。
故障现象：无法打开 PDF/Office 文件；检测：是否安装对应软件，文件是否损坏，软件是否兼容；修复：安装 Adobe Reader/Office，修复损坏文件，安装兼容版本软件。
故障现象：系统自动弹出广告，弹窗过多；检测：是否安装恶意软件，浏览器是否被劫持；修复：卸载恶意软件，重置浏览器设置，安装广告拦截插件。
故障现象：系统时间和日期错误；检测：主板电池是否没电，是否开启自动同步；修复：更换主板电池，开启自动同步时间（设置 - 时间和语言 - 自动设置时间）。
故障现象：管理员权限丢失，无法操作；检测：账户是否被降级，系统是否故障；修复：切换管理员账户，重建管理员账户，修复系统。
故障现象：系统无法更新，提示错误；检测：网络是否正常，更新服务是否开启，是否有残留更新文件；修复：检查网络，开启 Windows Update 服务，清理更新缓存。
故障现象：软件打开后显示乱码；检测：软件是否兼容系统，区域语言设置是否正确；修复：安装兼容版本，设置区域语言为中文（中国）。
故障现象：系统休眠后无法唤醒，只能强制重启；检测：电源管理驱动是否过时，休眠设置是否错误；修复：更新电源管理驱动，关闭快速启动，调整休眠设置。
故障现象：无法删除文件，提示 “文件正在使用”；检测：是否有程序占用文件，是否为系统文件；修复：关闭占用程序，重启电脑后删除，确认非系统文件后强制删除。
故障现象：系统托盘图标缺失；检测：系统是否卡顿，图标缓存是否损坏；修复：重启资源管理器，重建图标缓存（删除 C:\Users\ 用户名 \AppData\Local\IconCache.db）。
故障现象：浏览器无法打开网页，提示 “无法解析域名”；检测：DNS 是否设置错误，网络是否正常；修复：设置 DNS 为 223.5.5.5，检查网络连接，重启浏览器。
故障现象：系统提示 “磁盘空间不足”；检测：C 盘是否满溢，是否有大文件占用；修复：清理 C 盘垃圾，卸载不常用软件，转移大文件至其他盘。
故障现象：软件快捷键无法使用；检测：快捷键是否冲突，软件是否兼容，驱动是否正常；修复：关闭冲突软件，安装兼容版本，更新键盘驱动。
五、硬件其他故障（20 条）
故障现象：键盘部分按键失灵；检测：按键是否进液，排线是否松动，驱动是否损坏；修复：清理键盘进液（断电晾干），重新插拔排线，更新键盘驱动。
故障现象：触控板漂移，点击不准确；检测：触控板是否有灰尘，驱动是否过时，是否开启手势；修复：用干布擦拭触控板，更新触控板驱动，关闭不必要手势。
故障现象：屏幕亮度无法调节；检测：快捷键是否有效，电源管理驱动是否正常，显卡驱动是否过时；修复：使用 Fn+F1/F2 调节，更新电源管理和显卡驱动。
故障现象：笔记本摄像头无法使用；检测：摄像头是否被禁用，驱动是否过时，是否有物理遮挡；修复：在设备管理器启用摄像头，更新驱动，移除遮挡物。
故障现象：摄像头画面模糊；检测：镜头是否有灰尘，摄像头是否损坏，光线是否不足；修复：用干净布擦拭镜头，改善光线条件，联系售后检测摄像头。
故障现象：笔记本扬声器声音小，有杂音；检测：音量是否调至最大，扬声器是否堵塞，驱动是否过时；修复：调大音量，清理扬声器灰尘，更新音频驱动。
故障现象：光驱无法读取光盘；检测：光盘是否损坏，光驱是否有灰尘，驱动是否正常；修复：更换完好光盘，清理光驱激光头，更新光驱驱动。
故障现象：笔记本读卡器无法识别 SD 卡；检测：SD 卡是否损坏，读卡器是否有灰尘，驱动是否过时；修复：测试 SD 卡在其他设备，清理读卡器灰尘，更新读卡器驱动。
故障现象：电池鼓包，机身变形；检测：是否过充，电池是否老化；修复：立即停止使用，更换原装电池，避免安全隐患。
故障现象：笔记本接口松动，接触不良；检测：接口是否有损坏，是否频繁插拔；修复：减少频繁插拔，联系售后更换松动接口。
故障现象：笔记本屏幕闪烁，不稳定；检测：显卡驱动是否过时，屏幕是否损坏，电源是否稳定；修复：更新显卡驱动，外接显示器排查，联系售后检测屏幕。
故障现象：笔记本合盖后无法休眠；检测：电源设置是否正确，驱动是否正常；修复：在电源设置中设置 “合盖休眠”，更新电源管理驱动。
故障现象：笔记本指示灯异常闪烁；检测：对照说明书查看指示灯代码，是否有硬件故障；修复：按代码提示排查对应硬件（如电池、硬盘），重启电脑。
故障现象：笔记本外壳发热不均，局部烫手；检测：散热口是否堵塞，风扇是否故障；修复：清理散热口灰尘，更换故障风扇。
故障现象：笔记本 USB 接口供电不足，外接设备无法识别；检测：是否连接多个 USB 设备，接口是否老化；修复：减少同时连接的 USB 设备，更换供电足的接口。
故障现象：笔记本麦克风声音小，对方听不清；检测：麦克风音量是否调大，是否有遮挡，驱动是否正常；修复：调大麦克风音量，移除遮挡物，更新音频驱动。
故障现象：笔记本无法外接显示器，接口无反应；检测：接口是否损坏，视频线是否兼容，显卡是否故障；修复：更换接口和视频线，更新显卡驱动，联系售后检测。
故障现象：笔记本键盘背光无法开启；检测：快捷键是否有效，驱动是否过时，是否支持背光；修复：使用 Fn+F4 调节，更新键盘驱动，确认机型支持背光。
故障现象：笔记本硬盘异响，读取速度慢；检测：硬盘是否有坏道，是否老化；修复：用 DiskGenius 检测并修复坏道，备份数据更换硬盘。
故障现象：笔记本运行时发出 “滴滴滴” 报警声；检测：查看 BIOS 报警代码，排查内存、显卡、硬盘故障；修复：拔插内存，检测硬盘，联系售后排查对应硬件。
