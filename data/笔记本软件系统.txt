系统基础设置类
Windows 11 系统激活：预装机型开机自动激活，重装后通过数字许可证自动激活。
系统语言设置：Windows→设置→时间和语言→语言和区域；Mac→系统设置→通用→语言与地区。
更改系统日期和时间：开启 “自动设置时间和时区”，避免手动调整出错。
桌面图标显示设置：Windows→设置→个性化→主题→桌面图标设置，勾选常用图标。
任务栏位置调整：Windows 可拖至顶部  左侧  右侧，Mac 仅支持底部，右键任务栏设置。
系统声音设置：调整音量、设置默认音频设备，通过 “声音控制面板” 配置麦克风  扬声器。
屏幕分辨率调整：Windows→设置→系统→显示；Mac→系统设置→显示器，选择适配分辨率。
关闭系统自动更新：Windows→服务→Windows Update→禁用；Mac→系统设置→通用→软件更新→关闭自动更新。
启用系统防火墙：阻挡恶意网络连接，Windows 和 Mac 默认开启，无需手动操作。
系统休眠  睡眠设置：Windows→电源选项；Mac→电池，设置无操作后的休眠时间。
系统优化类
清理系统垃圾：Windows 使用 “磁盘清理”，Mac 使用 “存储管理→优化存储”。
禁用开机自启程序：Windows→任务管理器→启动；Mac→系统设置→通用→登录项。
关闭后台应用：Windows→设置→隐私和安全性→后台应用；Mac→系统设置→通用→后台应用。
调整虚拟内存大小：Windows→系统属性→高级→性能→设置→高级→虚拟内存，建议为物理内存的 1.5 倍。
开启硬件加速：浏览器→设置→系统和性能，启用硬件加速，降低 CPU 占用。
清理 DNS 缓存：Windows→CMD（管理员）→ipconfig flushdns；Mac→终端→sudo dscacheutil -flushcache。
关闭系统动画效果：Windows→设置→辅助功能→视觉效果；Mac→系统设置→辅助功能→显示→降低动态效果。
优化磁盘碎片：HDD 硬盘定期整理，Windows→磁盘碎片整理和优化；SSD 无需碎片整理。
关闭远程协助：Windows→设置→系统→远程桌面；Mac→系统设置→通用→共享，避免远程控制风险。
卸载系统预装软件：Windows→控制面板→程序和功能；Mac→应用程序→右键删除，清理残留文件。
软件管理类
软件安装路径：默认安装在 C 盘，自定义安装时选择 D 盘，避免占用系统盘空间。
软件下载渠道：优先选择官网下载，避免第三方平台的捆绑软件和恶意程序。
软件版本选择：稳定版适合日常使用，测试版可能有 bug，仅开发  测试场景使用。
批量卸载软件：使用 IObit Uninstaller（Windows）、CleanMyMac（Mac），彻底清理残留文件。
软件更新设置：重要软件（如办公、杀毒）开启自动更新，小众软件手动更新。
绿色软件使用：无需安装，解压即可运行，避免修改系统注册表，适合临时使用。
软件兼容性设置：右键软件图标→属性→兼容性，选择兼容的系统版本，解决旧软件无法运行问题。
软件权限管理：Windows→设置→隐私和安全性；Mac→系统设置→隐私与安全性，控制软件访问权限。
浏览器插件管理：定期清理无用插件，仅保留常用插件（如广告拦截、密码管理）。
软件备份与恢复：重要软件（如编程 IDE）导出配置文件，重装系统后快速恢复。
系统故障排查类
系统蓝屏：记录蓝屏代码（如 0x0000007B），通过搜索引擎查找解决方案，多为驱动  硬件问题。
系统卡顿：检查 CPU  内存  硬盘占用，关闭高占用程序，清理系统垃圾，优化启动项。
无法联网：更新网卡驱动，检查网络设置，重启路由器，重置网络配置（Windows→CMD→netsh winsock reset）。
软件无法打开：检查软件兼容性，更新软件版本，重新安装软件，清理缓存文件。
系统无法启动：使用 WindowsPE 启动盘修复，或通过系统还原点回滚，严重时重装系统。
驱动异常：使用 Driver Booster（Windows）、MacOS 自动更新，更新显卡  主板  声卡等驱动。
桌面图标消失：Windows→视图→显示桌面图标；Mac→终端→killall Finder，重启 Finder。
系统提示缺少.dll 文件：从官网下载对应.dll 文件，放入 System32 文件夹，或重新安装相关软件。
输入法无法切换：Windows→设置→时间和语言→语言和区域→高级键盘设置；Mac→系统设置→键盘→输入法。
系统自动重启：检查电源设置，关闭自动重启，排查硬件过热、驱动异常问题。
安全防护类
安装正规杀毒软件：如 360 安全卫士、火绒、Mac 版腾讯柠檬，定期全盘扫描。
开启 Windows Defender：Windows 自带杀毒软件，实时防护恶意软件，无需额外安装杀毒软件。
定期更新系统补丁：修复安全漏洞，Windows→设置→Windows 更新；Mac→系统设置→软件更新。
禁用 U 盘自动运行：防止 U 盘病毒入侵，Windows→组策略→计算机配置→管理模板→系统→关闭自动播放。
设置复杂登录密码：包含字母、数字、符号，定期更换，避免简单密码被破解。
开启文件加密：Windows→右键文件→属性→高级→加密内容以保护数据；Mac→磁盘工具→加密宗卷。
谨慎打开陌生邮件附件：避免钓鱼邮件和恶意软件，优先扫描后再打开。
清理浏览器缓存和 Cookie：保护隐私，避免浏览记录和账号信息泄露。
关闭未知端口：通过防火墙限制不必要的端口开放，减少黑客攻击风险。
定期备份重要数据：使用 U 盘、移动硬盘或云盘，避免病毒、系统崩溃导致数据丢失。
实用工具类
压缩解压工具：WinRAR、7-Zip（Windows），The Unarchiver（Mac），支持多种压缩格式。
截图工具：Snipaste（Windows），Skitch（Mac），支持截图编辑、标注功能。
视频播放工具：PotPlayer（Windows），IINA（Mac），支持多种格式，播放流畅。
办公软件：Microsoft Office、WPS，包含 Word、Excel、PPT，满足日常办公需求。
文件管理工具：Total Commander（Windows），Path Finder（Mac），增强文件管理功能。
密码管理工具：1Password、Bitwarden，安全存储账号密码，避免遗忘。
远程控制工具：TeamViewer、向日葵，远程协助他人或控制自己的电脑。
系统清理工具：CCleaner（Windows），CleanMyMac（Mac），清理垃圾、优化系统。
编程工具：Visual Studio Code、PyCharm，支持多种编程语言，轻量高效。
图片编辑工具：Photoshop、GIMP（免费），满足图片裁剪、修图等需求。
快捷键使用类
Windows 常用快捷键：Win+D 显示桌面，Win+E 打开文件资源管理器，Ctrl+Shift+Esc 打开任务管理器。
Mac 常用快捷键：Cmd+Space 打开聚焦搜索，Cmd+Tab 切换应用，Cmd+W 关闭窗口。
文本编辑快捷键：CtrlCmd+C 复制，CtrlCmd+V 粘贴，CtrlCmd+Z 撤销，CtrlCmd+S 保存。
浏览器快捷键：CtrlCmd+T 新建标签页，CtrlCmd+W 关闭标签页，CtrlCmd+F 查找内容。
截图快捷键：Windows→Win+Shift+S，Mac→Cmd+Shift+4，快速截取屏幕。
窗口操作快捷键：Windows→Win + 左  右箭头分屏，Win + 上箭头最大化，Win + 下箭头最小化。
音量调节快捷键：Fn+F11 降低音量，Fn+F12 调高音量，Fn+F10 静音。
键盘背光快捷键：Fn+F3 开启  关闭，Fn+F2F4 调节亮度（不同品牌快捷键不同）。
锁屏快捷键：Windows→Win+L，Mac→Cmd+Ctrl+Q，快速锁定屏幕。
任务管理快捷键：Ctrl+Alt+Del 打开任务管理器，强制结束无响应程序。
网络设置类
连接 WiFi：Windows→右下角网络图标→选择 WiFi→输入密码；Mac→右上角 WiFi 图标→选择 WiFi。
忘记 WiFi 密码：Windows→设置→网络和 Internet→WiFi→管理已知网络；Mac→系统设置→网络→WiFi→详细信息→忘记此网络。
设置静态 IP：Windows→网络属性→IPv4 属性；Mac→网络→高级→TCPIP，手动设置 IP 地址。
共享网络：Windows→移动热点；Mac→共享→互联网共享，将网络共享给其他设备。
修复网络连接：Windows→设置→网络和 Internet→高级网络设置→网络重置；Mac→系统设置→网络→高级→TCPIP→续租 DHCP 租约。
更改 DNS 服务器：Windows→网络属性→IPv4 属性；Mac→网络→高级→DNS，添加公共 DNS（如 8.8.8.8）。
查看 IP 地址：Windows→CMD→ipconfig；Mac→终端→ifconfig，查看本机 IP 和网关。
测试网络连通性：Windows→CMD→ping 223.5.5.5；Mac→终端→ping 223.5.5.5，检测网络是否通畅。
开启 VPN：Windows→设置→网络和 Internet→VPN；Mac→系统设置→网络→VPN，添加 VPN 连接。
关闭网络代理：Windows→设置→网络和 Internet→代理；Mac→系统设置→网络→代理，避免影响网络速度。
多系统与虚拟机类
安装双系统（Windows+Mac）：需确认笔记本支持，通过 Boot Camp（Mac）或 U 盘启动盘安装。
切换双系统：开机时按 Option（Mac）或 F12（Windows），选择要启动的系统。
虚拟机软件：VMware Workstation、VirtualBox（WindowsMac），在现有系统中运行其他系统。
虚拟机配置：分配合理的内存（建议为物理内存的 12）和 CPU 核心，避免占用过多资源。
虚拟机网络设置：选择 “桥接模式” 可让虚拟机接入局域网，“NAT 模式” 通过主机网络上网。
虚拟机文件共享：设置共享文件夹，实现主机和虚拟机之间的文件传输。
虚拟机快照：创建快照保存当前系统状态，后续可恢复到该状态，适合测试场景。
双系统磁盘分区：预留足够空间（≥50GB）安装第二个系统，避免空间不足。
虚拟机性能优化：关闭虚拟机后台程序，启用硬件加速，提升运行速度。
双系统引导修复：引导文件损坏时，使用 EasyBCD（Windows）或 Disk Utility（Mac）修复。
系统备份与恢复类
创建系统还原点：Windows→控制面板→恢复→创建还原点，系统故障时回滚。
制作系统镜像：Windows→控制面板→备份和还原→创建系统映像；Mac→时间机器，备份整个系统。
U 盘启动盘制作：使用 Rufus（Windows）、Etcher（Mac），制作 WindowsMac 启动盘，用于重装系统。
系统重置：Windows→设置→系统→恢复→重置此电脑；Mac→系统设置→通用→传输或还原 Mac→抹掉所有内容与设置。
数据备份工具：Windows→文件历史；Mac→时间机器，自动备份重要文件。
云备份：使用 OneDrive、百度云盘，同步文件到云端，跨设备访问，防止本地数据丢失。
系统恢复介质：创建恢复 U 盘  光盘，系统无法启动时用于修复，通过厂商工具制作。
选择性恢复：仅恢复特定文件  文件夹，不影响其他数据，通过备份工具选择恢复内容。
重装系统前备份：备份桌面、文档、下载文件夹等个人数据，避免丢失。
系统恢复后设置：安装必备软件、恢复网络设置、导入备份文件，快速恢复使用状态。
