CPU 核心知识类
CPU 是笔记本的 “大脑”，负责处理所有运算任务，核心数和主频是关键参数。
核心数越多，多任务处理能力越强，如 8 核 16 线程适合视频剪辑、编程开发。
主频（基础频率）越高，单核性能越强，影响软件启动速度、游戏帧率。
睿频是 CPU 的动态加速技术，负载高时自动提升频率，如 i7-14700H 基础 2.7GHz，睿频 5.4GHz。
Intel 和 AMD 是主流 CPU 厂商，Intel 稳定性强、兼容性好，AMD 性价比高、多核性能突出。
移动端 CPU 分为低功耗 U 系列（15W，适合轻薄本）和高性能 H 系列（45W，适合游戏本）。
CPU 缓存（L1/L2/L3）越大，数据读取速度越快，多任务处理更流畅。
制程工艺越先进（如 3nm/4nm），CPU 功耗越低、发热越小，性能更强。
超频是通过软件提升 CPU 频率，仅适合高端型号，普通用户不建议操作，可能损坏硬件。
CPU 虚拟化技术（VT-x/AMD-V）支持虚拟机运行，编程开发、测试场景必备。
GPU 核心知识类
GPU（显卡）负责图形处理，分为集成显卡和独立显卡，影响游戏、设计软件流畅度。
集成显卡内置在 CPU 中，功耗低、无独立显存，适合办公、视频播放。
独立显卡有独立显存和核心，性能强，分为游戏显卡（NVIDIA RTX/AMD RX）和专业图形卡（NVIDIA Quadro/AMD FirePro）。
显存容量越大，处理高清视频、3D 模型的能力越强，游戏本建议≥8GB，设计本≥12GB。
显存位宽影响数据传输速度，≥128bit 为入门级，≥256bit 为中高端。
NVIDIA DLSS 和 AMD FSR 是超分技术，可提升游戏帧率，不损失太多画质。
光追技术（Ray Tracing）模拟真实光线效果，3A 游戏玩家优先选择支持光追的显卡。
独显直连技术让显卡直接输出画面，游戏帧率比混合模式高 10%-30%。
显卡功耗越高，性能释放越充分，游戏本显卡功耗≥140W，轻薄本独显≤35W。
专业图形卡针对设计软件（AutoCAD、Pr）优化，渲染速度比游戏卡快，普通用户无需追求。
内存核心知识类
内存（RAM）是临时存储数据的空间，容量越大，可同时运行的软件越多。
日常办公建议≥16GB，游戏 / 设计 / 编程建议≥32GB，避免内存不足导致卡顿。
内存频率越高，数据传输速度越快，DDR5 4800MHz 比 DDR4 3200MHz 性能提升明显。
双通道内存（2×8GB/2×16GB）比单通道（1×16GB）带宽高 1 倍，核显机型对双通道依赖更高。
内存时序（如 CL40）越低，延迟越小，运行速度越快，对游戏和多任务影响显著。
板载内存无法升级，购买时需确认容量是否足够，支持拓展的机型预留内存插槽。
内存品牌优先选三星、海力士、镁光等原厂颗粒，兼容性和稳定性更优。
DDR5 内存比 DDR4 更节能，频率更高，是新一代笔记本的主流配置。
内存电压分为标准电压（1.2V）和低电压（1.1V），低电压更省电，适合轻薄本。
内存拓展时需选择同频率、同规格的内存，避免降频运行，影响性能。
硬盘核心知识类
硬盘是永久存储数据的设备，分为 SSD（固态硬盘）和 HDD（机械硬盘）。
SSD 读写速度比 HDD 快 10 倍以上，开机、软件启动速度显著提升，优先选择。
SSD 接口分为 SATA 和 M.2，M.2 接口支持 PCIe 4.0 协议，读写速度≥3500MB/s。
硬盘容量：系统盘≥512GB，存储需求大（视频 / 游戏）建议≥1TB。
SSD 闪存类型：TLC 闪存寿命≥600TBW，QLC 闪存寿命较短，适合存储盘。
缓存容量越大，SSD 连续写入速度越稳定，避免小容量缓存耗尽后掉速。
HDD 适合存储大量数据，价格低、容量大，但抗震性差、噪音大，不适合移动场景。
硬盘健康状态可通过 CrystalDiskInfo 检测，关注 “剩余寿命”“坏道” 等参数。
NVMe 协议是 M.2 SSD 的高速传输协议，比 SATA 协议快 3 倍以上。
双硬盘组合（SSD+HD）兼顾速度和容量，SSD 装系统和常用软件，HDD 存文件。
屏幕核心知识类
屏幕尺寸：13-14 英寸便携，15.6-16 英寸均衡，17.3 英寸大屏沉浸感强。
分辨率：1080P（1920×1080）为入门，2.5K（2560×1440）为中端，4K（3840×2160）为高端。
刷新率：60Hz 适合办公，144Hz/165Hz 适合游戏，高刷屏幕画面更流畅。
色域覆盖：100% sRGB 适合办公，100% DCI-P3 适合设计 / 影视，色域不足会导致色彩失真。
屏幕材质：IPS 屏可视角度广（178°），色彩均匀；TN 屏可视角度差，价格低。
亮度：≥300nits 日常使用足够，≥400nits 适合户外，HDR 屏峰值亮度≥500nits。
响应时间：游戏本≤3ms，避免动态模糊；办公本≤15ms 即可。
色准 ΔE≤2 为专业级，适合设计；ΔE≤3 满足日常使用，色准差会导致颜色偏差。
OLED 屏色彩鲜艳、对比度高，但寿命较短（3-5 年），易烧屏，适合影视 / 设计。
屏幕调光：DC 调光无频闪，对眼睛友好；PWM 调光低亮度有频闪，敏感人群需避开。
电池核心知识类
笔记本电池多为锂聚合物电池，比锂电池更轻薄、安全性更高。
电池容量用 Wh（瓦时）表示，数值越大续航越强，轻薄本≥56Wh，游戏本≥80Wh。
电池循环寿命：≥1000 次循环后容量衰减≤20%，正常使用可维持 2-3 年。
过充保护：现代笔记本电池充满后自动断电，不会过度充电，可放心插电使用。
电池校准：每 3-6 个月一次，充满电后继续插电 1 小时，再放电至 20%，提升电量显示准确性。
高温 / 低温环境会影响电池寿命，避免在≥40℃或≤0℃环境下长时间使用。
电池鼓包是危险信号，可能导致机身变形、爆炸，需立即停止使用并更换。
可拆卸电池已基本淘汰，不可拆卸电池更换需联系售后，避免自行拆解。
长期插电使用时，开启电池充电阈值（如充电至 80% 停止），延长电池寿命。
第三方电池质量参差不齐，更换时优先选择原厂或正规品牌电池。
散热系统知识类
散热系统由风扇、热管、散热鳍片组成，负责排出 CPU 和 GPU 产生的热量。
双风扇 + 4 热管是中高端游戏本的标配，散热鳍片面积越大，散热效果越好。
散热硅脂填充在 CPU/GPU 与热管之间，传导热量，2-3 年需更换一次，避免干涸影响散热。
风扇转速可通过厂商工具调节，高负载时调高转速，低负载时调低降噪。
散热孔堵塞会导致过热降频，定期用压缩空气罐吹扫，清理灰尘。
液态金属散热效果比硅脂好，但导电性强，安装不当可能短路，仅高端机型使用。
散热模组材质：铜热管导热性优于铝热管，是主流选择。
过热降频是 CPU/GPU 的自我保护机制，温度超过 95℃时自动降低频率，避免损坏。
轻薄本散热模组较小，不适合长时间高负载运行，否则会频繁降频。
散热风扇噪音过大可能是灰尘过多或轴承老化，清理灰尘无效则需更换风扇。
接口与外设知识类
USB-A 接口是传统接口，兼容 U 盘、鼠标等外设，USB 3.2 Gen2 传输速度 10Gbps。
USB-C/Type-C 接口支持充电、数据传输、视频输出，一线多用，是现代笔记本主流接口。
HDMI 接口用于外接显示器 / 电视，HDMI 2.1 支持 4K 120Hz 输出，HDMI 2.0 支持 4K 60Hz。
Thunderbolt 4 接口传输速度 40Gbps，支持外接显卡坞、4K 双屏输出，专业用户必备。
3.5mm 耳机接口兼容耳机 / 音箱，无接口机型需使用 Type-C 转接器。
RJ45 网口（千兆 / 2.5G）适合固定办公场景，传输速度稳定，轻薄本常省略。
SD 读卡器适合摄影用户，UHS-II 规格读写速度≥300MB/s，普通用户无需强制要求。
接口插拔寿命：Type-C≥10000 次，USB-A≥5000 次，避免频繁插拔损坏接口。
拓展坞可扩展接口数量，适合轻薄本接口不足的场景，选择时匹配笔记本接口规格。
外接显示器时，需确认笔记本接口和显示器接口是否一致，不一致需使用转接器。
主板与其他硬件类
主板是笔记本硬件的 “连接中枢”，所有硬件都通过主板连接，影响扩展性和稳定性。
主板芯片组（如 Intel Z790、AMD B650）决定 CPU 兼容性、内存频率上限、接口数量。
BIOS/UEFI 是主板固件，负责启动系统，可通过厂商官网更新，修复漏洞、提升兼容性。
声卡影响音质输出，支持 Hi-Res Audio 认证的机型，搭配高品质耳机音质更优。
网卡分为有线网卡和无线网卡，WiFi 6/6E 网卡传输速度比 WiFi 5 快 2 倍以上。
蓝牙 5.0 及以上版本支持远距离传输、多设备连接，适合连接耳机、鼠标等外设。
摄像头分辨率≥1080P 为高清，支持 Windows Hello / 面容 ID 的机型，解锁更便捷。
麦克风降噪功能可减少环境噪音，视频会议场景更实用，通过软件开启。
指示灯可显示电源、电池、硬盘、WiFi 等状态，便于快速判断设备运行情况。
机身材质：铝合金质感好、耐摔，塑料机身性价比高，碳纤维机身轻便但价格贵。
硬件故障排查类
笔记本无法开机，电源指示灯不亮：检查电源适配器是否通电，接口是否松动。
开机黑屏：可能是屏幕故障或显卡问题，外接显示器测试，有显示则是屏幕故障。
蓝屏重启：多为内存不兼容、驱动异常，检查内存兼容性，更新显卡 / 主板驱动。
硬盘无法识别：检查硬盘接口是否松动，用 CrystalDiskInfo 检测硬盘健康状态，有坏道需更换。
风扇不转：可能是风扇故障或主板供电问题，立即停止使用，避免过热损坏 CPU/GPU。
键盘部分按键失灵：清理键帽缝隙异物，无效则是键盘排线松动或键盘故障，需维修。
电池不充电：检查电源适配器是否匹配，电池是否鼓包，驱动是否异常。
网络无法连接：更新网卡驱动，检查 WiFi 密码，路由器重启，排除网络环境问题。
屏幕闪烁：可能是屏幕排线松动或显卡驱动问题，重新插拔排线（需专业人员），更新驱动。
触控板失灵：重启电脑，更新触控板驱动，检查是否误关触控板（Fn + 触控板键）。
